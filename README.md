## Installation

1.    cd ~
2.    curl -sL https://deb.nodesource.com/setup_8.x -o nodesource_setup.sh
3.    sudo bash nodesource_setup.sh
4.    sudo apt-get install nodejs

Check your versions by
nodejs -v
npm -v
